# 题目：
编写一个算法来判断一个数是不是“快乐数”。
一个“快乐数”定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。如果可以变为 1，那么这个数就是快乐数。
# 程序：
~~~
bool isHappy(int n) {
        while(n!=1&&n!=4)
        {
            int res=0;
            while(n)
            {
                res+=(n%10)*(n%10);
                n=n/10;
            }
            n=res;            
        }
        return n==1;
    }
~~~
# 心得：
非快乐数有个特点，循环的数字中必定会有4，利用这个规律，根据快乐数的规律，即可判断出是否为快乐数。
